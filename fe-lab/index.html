⌨ 键盘浏览指南

在本页面:

✅ 按Enter键可关闭本指南

✅ 按Ctrl + Enter键可重新显示本指南

✅ 按上/下方向键可切换侧边导航项目

✅ 按左/右方向键可切换分页

✅ 按Home/End键可直接跳转到第一页/最后一页

✅ 按1-6数字键(横着排的那种，不是小键盘的数字键)可直接进入页面上相应顺序的特效详情页

✅ 按L键可在浅色/深色模式之间切换

在特效详情页:

✅ 按左/右方向键可展开/收起代码框

✅ 按Shift + H/C/J键可分别复制 HTML / CSS / JavaScript 代码
